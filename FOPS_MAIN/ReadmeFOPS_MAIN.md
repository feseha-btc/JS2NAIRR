## This folder contains the tar.gz compressed file of the tool.
## When decompressed, it will have the
  1. The code_runner_FOPS.sh wrapper script that will call the necessary scripts in their order to generate fuzzy oligos.
  2. the data folder which contains example sequence and sequence name file
  3. scripts folder with the scripts listed below in it:
    * Fops_banchmark_sampler.py
   
    * Fops_geminiSimpleCluster_GPU.py
   
    * Fops_generate_oligo.py
   
    * Fops_generate_report.py
   
    * Fops_get_longest_sequence.py
   
    * Fops_rename_headers.sh
   
    * Fops_run_analysis.sh
